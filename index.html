Inflation Calculator

Find the value of rupee (INR) from any year in today's terms!

This calculator estimates the value of Indian Rupees over time based on historical inflation rates. Reliable data is available from 1960 onwards, and for years prior to 1960 (since 1947), the values are extrapolated based on similar trends. Inflation rates fluctuate yearly, so the calculation is an approximation.

Source: World Data - Inflation Rates
